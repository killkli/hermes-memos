# Entity Pages

Person pages, place pages, organization pages, and thing pages.

## Types of Entities

- **People** — Researchers, authors, developers, historical figures
- **Organizations** — Companies, labs, universities, open-source projects
- **Places** — Cities, countries, landmarks, venues
- **Products** — Software, hardware, books, publications
- **Models** — AI models, system architectures

## Page Format

Every entity page should include:
- Overview / what it is
- Key facts and dates
- Relationships to other entities ([[wikilinks]])
- Source references (to raw sources)
