# 台股晨報知識消化
**日期：** 2026-03-28

## 宏觀格局判斷

1. **伊朗封鎖荷姆茲海峽升級為系統性風險**：日均 1,100 萬桶原油受威脅，油價單日狂飆 7%，布蘭特觸及 $112 創 2022 年以來新高。麥格理警告若持續至 6 月底，油價可能飆至 $200。這是繼 2022 年之後再次出現的「黑天鵝式油價衝擊」，直接威脅全球供應鏈成本結構。

2. **台股「外资大逃殺」創歷史紀錄**：3 月以來累計賣超 8,235 億元，不只是單週撤離而是整月結構性拋售。這與 2022 年中共軍演期間的外資撤離模式不同——此次没有政治事件但有地緣戰爭邏輯，外資正在對冲「台灣被捲入美伊衝突」的尾部風險。

3. **記憶體/HBM 雙重結構性利空確認**：ITC 337 調查鎧俠、SK 海力士 + TurboQuant 新技術疑慮 + 美光自高點累跌 30%，記憶體族群在 2026 年初經歷暴起暴落後，正進入「擠泡」階段。

## 產業深度分析

### 一、半導體/AI

| 維度 | 分析 |
|------|------|
| **基本面** | 台積電美國封裝廠 Q2 動工（4 座先進封裝廠規劃），深化美國在地佈局方向確立，但 Q2 動工代表的是長期敘事而非短線催化劑。華為 950PR 晶片 4 月量產下半年全面出貨，與輝達 CUDA 相容性提高，這是中國 AI 晶片自主化的重要里程碑，但對台灣半導體供應商的實質影響仍待觀察。 |
| **技術面** | 美光股價從 3/18 財報後累跌 30%，技術面已進入標準熊市修正區間。SK 海力士赴美 ADR 申請代表記憶體產業正在重構上市地緣，長期對韓廠有利但短期稀釋效應待消化。 |
| **國際政治** | 美國對記憶體大廠發動 337 調查，顯示智慧財產權武器化已從華為擴展至整個記憶體供應鏈。台灣廠商（旺宏、南亞科等）若被波及，將面臨法律與市場的雙重風險。 |

### 二、金融/保險

| 維度 | 分析 |
|------|------|
| **基本面** | 投信逆勢買超金融股（台新新光金、中信金），顯示內資機構認為金融股在此波下跌中是相對安全的避風港。金控股的防禦屬性來自其固定收益取向，與科技股的高波動形成對比。 |
| **技術面** | 台股 3/27 成交量縮至 6,582 億元，人氣退潮明顯。在此環境下，金融股成交相對活絡，反映資金輪動至防禦性標的。 |
| **國際政治** | 中東戰事對金融業的影響是間接的——油價推升通膨 → Fed 降息延後 → 公債殖利率上行 → 壽險業的避險壓力增加。台灣壽險業持有大量海外债券，利率環境的不確定性是主要風險。 |

### 三、傳產/航運/能源 ⚠️

| 維度 | 分析 |
|------|------|
| **基本面** | 能源股（台塑化、台苯）因油價飆漲直接受益，但這是「成本推動型」的上漲而非需求拉動。中長期若油價压制制造业盈利，能源股的客户基础反而受损。 |
| **技術面** | 散裝和貨櫃航運受荷姆茲封鎖影響最大，但晨報資料有限，無法詳細分析。 |
| **國際政治** | 伊朗封鎖荷姆茲海峡若長期化，全球能源貿易路線被迫繞行好望角，航程增加 2-3 週，燃料成本大幅上升。葉門胡塞武裝已加入攻擊商船，紅海+荷姆茲雙重封鎖正在形成。 |

## 跨領域連結矩陣

```
半導體 ←→ 能源：中東油價上升 → 台灣製造業用電/天然氣成本上升 → 半導體廠商額外成本壓力
金融 ←→ 壽險：Fed 降息延後 → 壽險業避險成本上升 → 公債殖利率上行壓力
記憶體 ←→ 半導體：美光暴跌 30% → 南亞科、華邦電、旺宏等台灣記憶體廠商補跌壓力
航運 ←→ 能源：油價上漲直接增加航運燃料成本，即使運價上漲也可能被成本吃掉
台股 ←→ 外資：外资單月賣超 8,235 億是歷史級拋售，與地緣風險對冲需求直接相關
```
