# 台股晨報知識消化
**日期：** 2026-03-31

## 宏觀格局判斷

1. **「停戰狂歡」被伊朗官方打臉——市場定價失誤的典型教材**：3/30 台股大漲 3.58% 是基於川普單方面的「伊朗提出停火」說法；4/1 伊朗否認後美股期貨立刻由升轉跌。這是 2026 年以來最典型的「政策噪音」干擾市場案例，也顯示川普團隊在資訊發布上存在主動管理市場預期的操作。

2. **台股 3 月重挫近 13% —— 2022 年 9 月以來最惨單月**：主因是伊朗戰事升級 → 油價飆升 → 停滯性通膨疑慮。iShares MSCI 台灣 ETF（EWT）流出約 110 億美元創歷史最大，這不只影響台股流動性，也代表國際投資人正在對台灣地緣風險溢價重新定價。

3. **費半 +6.24% 的隱含訊號**：在整體市場動盪中，費半是相對最強的板塊，代表在半導體超級週期敘事尚未被完全破壞之前的資金選擇。台積電法說會（4/1）是下一個關鍵驗證點。

## 產業深度分析

### 一、半導體/AI

| 維度 | 分析 |
|------|------|
| **基本面** | 台積電法說會（4/1）是本週最大事件。ADR 大漲 6.7% 反映市場對其基本面的信任，但伊朗否認停火的時機恰好在法說會前夕——若法說內容指向 Q2 展望下修，股價將面臨雙重打擊。記憶體（美光、SK 海力士、威騰）下跌放緩但未止穩，TurboQuant + HBM 需求下滑的雙重利空尚待消化。 |
| **技術面** | 費半 +6.24% 是技術性反彈而非趨勢反轉。台積電 ADR 在 3/30 大漲後，4/1 期貨翻黑顯示短線獲利了結壓力大。 |
| **國際政治** | 伊朗革命隊宣布 4/1 起對中東 18 家美企（微軟、谷歌、蘋果、英特爾、特斯拉、波音）展開報復——這是對美制裁的反制，但同時使美伊談判更加複雜。台灣不在報復名單上，但若美企在台灣的供應商受到波及，供應鏈將面臨間接影響。 |

### 二、能源/傳產

| 維度 | 分析 |
|------|------|
| **基本面** | 伊朗停戰預期使 Brent 從 $112 跌至 $104，但 5 月合約仍有 $118，月線仍大漲 64%。美國汽油均價已突破 $4/加侖（2022 年以來首見），對消費者支出形成直接擠壓。科威特油輪遭伊朗攻擊顯示封鎖行動正在擴大而非收縮。 |
| **技術面** | 油價日線回落但月線仍超漲——這是典型的「黑天鵝消退初期的價格結構」，市場需要更多時間確認供給是否真的恢復。 |
| **國際政治** | 尼坦雅胡堅持持續轟炸，美防長赫格塞斯表示「未來數天是戰爭決定性時刻」——這與川普聲稱的「2-3 週內撤軍」形成公開矛盾，代表美國政府內部對伊政策存在分歧。 |

### 三、金融/保險

| 維度 | 分析 |
|------|------|
| **基本面** | 巴菲特「小額買進」的訊息對市場情緒有穩定作用，但這是「信心喊話」而非具體策略。伊朗革命隊 4/1 起對美企展開報復，若波及金融機構（信用卡、保險），可能觸發新一輪拋售。 |
| **技術面** | VIX 從高點回落至 ~23 水準，但絕對值仍在悲觀區域。VIX 回到正常水準（<20）需要地緣局勢明確降級。 |
| **國際政治** | 11 國發聯合聲明籲維護荷姆茲航運暢通——這是外交層面的支持，但沒有實質軍事行動配合象徵意義大於實質。 |
