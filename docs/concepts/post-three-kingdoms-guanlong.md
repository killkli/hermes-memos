# 後三國時代與關隴集團

## 後三國：政權繼承序列

傳統三國時代終結於晉朝統一，但「後三國」指的是一組延續至唐代的政權序列：

**西魏 → 北周 → 隋 → 唐**

這一序列並非斷裂的改朝換代，而是一個高度連續的統治集團在不同旗號下的執政。 北周禪讓於隋，隋禪讓於唐——表面是禪讓，實質是同一軍事貴族聯盟的政權延續。

## 關隴集團

關隴集團是理解這段歷史連續性的關鍵概念：

- **起源**：北魏六鎮之一——**武川鎮**（今內蒙古境內）的軍事貴族網絡
- **形成**：爾朱榮之亂後，武川軍閥賀拔岳、宇文泰等人入關建立西魏，形成關隴軍事貴族聯盟
- **特徵**：跨越朝代的政治聯盟，核心成員家族（如宇文泰家族、李虎家族、獨孤家族等）歷經西魏、北周、隋、唐四朝始終掌控核心權力

## 沒有純粹漢族王朝

這是理解這段歷史最重要的顛覆性視角：

- 關隴集團成員多具有**內亞鮮卑/突厥血統**（如宇文泰鮮卑姓「普屯氏」，李虎家族有鮮卑背景）
- 文化認同上高度雜糅——既不完全是漢族也不是純粹鮮卑，而是**兼具草原軍事傳統與漢魏制度遺產**的混合文化
- 隋唐皇室本身即出自關隴集團，楊堅有鮮卑普六茹氏背景，李淵家族亦有鮮卑血統
- 所謂「漢族王朝」是後世建構，與歷史實際有相當落差

## 從北周到隋：統一天下的政治連續性

北周滅北齊統一天下（577年），隨即由隋朝（楊堅）取代。隋滅陳（589年）完成統一。這個進程的連續性表現在：

- 政策與制度層面高度沿襲（均田制、府兵制、三省制等）
- 核心統治集團基本未變
- 隋煬帝的雄心（如大運河、征高句麗）是北周以來統一戰略的延續

> 理解這段歷史的關鍵：**不是朝代更替，而是同一政治軍事集團在不同歷史階段的執政形態**。
