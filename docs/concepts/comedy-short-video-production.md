# 邵氏武俠風搞笑短片泛化技能

## 概述

2026-05-05 session 中，將原本的英文諧音梗專用技能重構為泛用搞笑短片技能。經專業子代理人兩輪迭代評估後定版。

## 技能演進

```
英文諧音梗專用技能 → 泛用搞笑短片技能（邵氏武俠風）
```

- 原技能限縮於特定格式，重構後支援 9 種短片格式
- 保留邵氏武俠視覺風格作為核心美學基調

## Fast Intake Mode

採用 4 題選項式快速收集需求：

1. **格式選擇**：從 9 種格式中選一
2. **主角設定**：角色數量與類型
3. **搞笑方向**：喜劇機制偏好
4. **時長偏好**：15s / 30s / 60s

## 9 種短片格式分類

- 武打對決
- 追逐戲
- 密室逃脫
- 身分互換
- 時間穿越
- 江湖騙局
- 師徒過招
- 比武招親
- 群體混戰

## Comedy Mechanism Palette

| 機制 | 說明 |
|------|------|
| 反差 | 嚴肅場景突然出現荒謬元素 |
| 誤會 | 角色之間的認知落差 |
| 升級 | 衝突逐步失控升級 |
| 吐槽 | 第四面牆打破、自嘲 |
| 反轉 | 結局完全翻轉預期 |

> 喜劇機制可以疊加使用，但每支短片核心機制不超過 2 種，避免失焦。

## 固定輸出格式

```
需求摘要 → 腳本 → Prompt → 補充說明
```

1. **需求摘要**：一個段落總結核心創意
2. **腳本**：結構化場景腳本（含對白、動作指示）
3. **Prompt**：可直接餵入 [AI 影片生成實戰 — Seedance 2.0](/hermes-memos/concepts/ai-video-generation-seedance/) 的影片 prompt
4. **補充**：音效建議、替代方案、迭代方向

## 專業評估流程

- 經專業子代理人（扮演喜劇編劇顧問）進行兩輪迭代評估
- 評估維度：笑點密度、節奏控制、視覺可行性、受眾適配度
- 每輪評估後調整 Prompt 模板與輸出格式
