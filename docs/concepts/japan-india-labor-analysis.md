# 在日印度勞動力結構分析

**概念層級：** 分析案例 + 方法論反思
**入庫日期：** 2026-04-14
**相關原始來源：** `sources/2026-04-14_japan-india-labor-statistics.md`

---

## 1. 概述

在日印度勞動力是一個佔全體外國人約1.4%（2023年：46,262人；2024年估：54,000人）的小型外國人群體。外界常將其與「IT大國」印象掛鉤，但實際數據結構存在重大驗證鴻溝。

---

## 2. 已知事實（HIGH置信度）

| 事實 | 數據 | 來源 |
|-----|------|-----|
| 在日印度人總數（2023） | 46,262人 | 法務省統計 |
| 在日印度人估總數（2024） | 約54,000人 | 新聞估算 |
| 佔全體外國人比例 | 約1.4%（2023） | 計算值 |
| 特定技能中印度人比例 | <2%，未進前10 | 入管庁2024統計 |

---

## 3. 未知/不可驗證區域

**以下數據無法通過官方統計驗證，任何聲稱均屬 LOW 置信度：**
- 印度人的產業別分佈（IT/餐飲/零售/製造）
- 印度人的簽證別細項分佈
- 任何精確到小數點的百分比（如「IT佔30%」）

**原因：** 日本官方統計通常只公佈「全體外國人」或「主要國籍（越南/中國/菲/尼）」的交叉分佈表，印度作為少數群體，其國籍別細項通常不在公佈範圍內。

---

## 4. 偏誤識別框架（本案例產出）

本案例是多代理人事後審查的典型應用。識別到6種偏誤，分為三類：

### 4.1 原始報告已識別（正確）
| 偏誤 | 說明 |
|-----|------|
| Aggregation Error | 將「最大類別30%」等同於「全體」 |
| Category Mixing | 簽證類別與職業類別混淆 |
| Visibility Bias | 高曝光服務業造成錯覺 |

### 4.2 新識別（本案例）
| 偏誤 | 說明 |
|-----|------|
| Source Fabrication Bias | 聲稱基於官方統計，實為估算 |
| Double Aggregation | 群組數值強行湊合，超出100% |
| Confirmation Bias | 預設立場後選擇性援引數據 |

## 5. 方法論教訓

### 5.1 多代理人審查框架
```
Generator → 提取數據聲明，建立驗證矩陣
Critic   → 識別偏誤，發現邏輯漏洞
Researcher → 基於可驗證數據，產出修正結論
```

### 5.2 關鍵原則
**當數據鴻溝大於聲稱精確度時，應擴大不確定性而非縮小。**

本案例中，報告呈現精確到個位數百分比的「產業分佈」，但這些數字根本不存在於任何官方統計中。面對這種情況，正確的回應是：「數據無法驗證」，而非填補一個估算數字。

### 5.3 置信度應用
| 級別 | 定義 | 應用 |
|-----|------|-----|
| HIGH | 多源一致 | 可直接引用 |
| MEDIUM | 單源/估算 | 標註不確定性 |
| LOW | 未驗證 | 視為未知，拒絕引用 |
