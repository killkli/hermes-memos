# 晉室南渡與僑吳姓

晉室南渡並非單純正統傳承，而是政治博弈；僑吳姓問題：北方世家與南方土著的政治整合困境；王敦與司馬睿的政治角力；東晉建國是北方軍事力量與南方土著利益的妥協產物。
