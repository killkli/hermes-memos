# 沒有單一漢族理論

## 核心論點

- **沒有單一固定的漢族身份** — 漢族身份並非自古恆定不變的本質實體，而是歷史上持續演變的政治與文化建構。
- **孝文帝改革是內亞性延續而非漢化** — 孝文帝的漢化政策被重新詮釋為內亞帝國傳統的延續，而非對「漢族」的同化。
- **關隴集團無純粹漢族血統** — 關隴軍事貴族集團具有多元血統，無法被視為「純粹漢族」。
- **身份認同是政治構建非種族本質** — 族群認同是政治權力運作的產物，而非客觀的種族生物學分類。

## 歷史脈絡

此理論挑戰傳統線性史觀中「漢族」作為穩定主体的敘事。魏晉南北朝時期的族群互動、婚姻網絡、政治聯盟，使得「漢族」身份呈現高度流動性與建構性。
