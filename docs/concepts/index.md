# Concept Pages

Ideas, techniques, frameworks, methods, and theories.

## Types of Concepts

- **Techniques** — Methods, algorithms, approaches
- **Frameworks** — Conceptual frameworks, mental models
- **Theories** — Scientific theories, hypotheses
- **Domains** — Fields of study, research areas
- **Events** — Conferences, movements, historical events (as conceptual entities)

## Page Format

Every concept page should include:
- Definition / explanation
- Current state of knowledge
- Open questions or debates
- Related concepts ([[wikilinks]])
