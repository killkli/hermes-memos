# 台股晨報知識消化
**日期：** 2026-04-02

## 宏觀格局判斷

1. **台股「史上第二大漲點」掩蓋了結構性脆弱**：4/1 台股大漲 4.58%（+1,451.83 點），成交量 6,173 億，三大法人同步買超合計 549.81 億。但這是在 3 月重挫 13% 基礎上的技術性反彈——跌幅越大反彈力道越猛，這是物理學而非基本面。關鍵是：這個反彈能否轉化為底部確認，還是只是「死貓跳」？

2. **外資從「歷史級賣超」逆轉為單日買超 232.51 億**：這是值得高度關注的訊號。外资在 3 月累計賣超 8,235 億後，4/1 出現單日買超——可能是空頭回補，也可能是新的長線佈局。無論如何，這代表外资對台股的悲觀情緒可能正在接近頂峰。

3. **面板和記憶體成交活絡的背後邏輯**：群創成交量 32.76 萬張成成交王、南亞科成交額突破 130 億——這不是基本面驅動的「價值發現」，而是被低估太久之後的資金輪動投機。面板的「光進銅退」和記憶體的「低位抄底」是兩種不同性質的資金行為，前者更具基本面基礎。

## 產業深度分析

### 一、半導體/AI

| 維度 | 分析 |
|------|------|
| **基本面** | 台積電大漲 95 元，ADR +6.78% 帶動費半 +2.82%。台積電仍是全球最確定的 AI 基礎設施供應商，在地緣風險環境下，市場傾向給予「確定性溢價」。鴻海、台達電同步反彈，顯示整個 AI 供應鏈的人氣正在恢復。輝達受惠投資邁威爾的消息激勵走強，但輝達在這個階段更多是「情緒標的」而非「基本面標的」。 |
| **技術面** | 費半是昨日相對抗跌的指數（+2.82%），在半導體庫存調整週期中，資金選擇「最終贏家」（TSMC）而非「整個族群」。這是技術性反彈中的典型分化。 |
| **國際政治** | 伊朗否認停火使地緣風險仍高，但市場選擇忽略短期噪音而聚焦中期趨勢——這個轉向是判斷是否真的觸底的重要觀察點。 |

### 二、金融/保險

| 維度 | 分析 |
|------|------|
| **基本面** | 凱基金（2883）法說會是今日焦點。壽險業在高利率環境下的利差收益正在改善，但 CSM 自律草案（見 04-29 晨報）指向監管風險正在上升——兩股力量同時存在。 |
| **技術面** | 投信和自營商在 4/1 的買超（合計 317 億）顯示內資機構在此點位願意承接。 |
| **國際政治** | 美銀警告伊朗戰事可能使油價全年維持 $100 以上——若成真，Fed 維持高利率更久，對金融機構的長期影響是「短空長多」（高利率等於高利差）。 |

### 三、面板/光電

| 維度 | 分析 |
|------|------|
| **基本面** | 群創成交量 32.76 萬張成成交王，友達同步亮眼。「光進銅退」是結構性趨勢——隨著資料中心對光纖傳輸需求上升，光學元件的戰略地位持續提升。但面板產業本身的供需週期仍在調整，產能過剩問題尚未完全解決。 |
| **技術面** | 面板股在 3 月大跌後出現報復性反彈，技術性因素大於基本面。投資人需區分「超跌反彈」和「基本面好轉」。 |

### 四、記憶體

| 維度 | 分析 |
|------|------|
| **基本面** | 華邦電、旺宏、南亞科擠進成交前五，宜鼎盤中亮燈漲停。但記憶體的基本面問題（TurboQuant + HBM 需求下滑）尚未解決，這是「技術性反彈」而非「週期性復甦」。 |
| **技術面** | 記憶體在 3 月經歷 30% 的暴跌後，低接買盤自然浮現。但「低接」不等於「到底」——記憶體景氣週期往往有「超跌 → 整理 → 二次測底 → 復甦」的路徑。 |

### 五、能源/傳產 ⚠️

美銀警告油價全年 $100 以上。高油價對台灣的影響：台灣高度依賴進口能源，油價維持高位對經常帳和製造業成本都是持續的逆風。

### 七、生技/醫療 ⚠️

### 八、綠能/電動車 ⚠️
