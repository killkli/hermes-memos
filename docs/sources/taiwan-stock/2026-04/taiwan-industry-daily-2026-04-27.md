# 台股晨報知識消化
**日期：** 2026-04-27（週末特刊）

## 宏觀格局判斷

1. **英特爾 Q1 EPS $0.29 大超預期，股價暴漲 23.6% 創 1987 年以來最大單日漲幅——這不只是「低谷反彈」，而是英特爾 IDM 2.0 策略的實質驗證**：英特爾的復甦如果持續，將成為台灣半導體供應鏈的潛在競爭對手（搶奪晶圓代工訂單），但也可能是客戶基礎擴大（為輝達之外的 AI 晶片設計公司提供代工）。

2. **輝達市值重回 5 兆美元——等於 25 個台積電**：輝達一家的市值等於全球大多數國家的 GDP，這個數字的膨脹速度比任何基本面都更快。但輝達目前的股價更多是「算力信仰」的期貨定價，而非當期財報的反映。

3. **費半連漲 18 個交易日創歷史新高——這是美國半導體的超級多頭，不是週期性復甦**：18 連漲在任何市場、任何時間框架中都是極端數據，暗示短期內可能出現急速回檔。

## 產業深度分析

### 一、半導體/AI

| 維度 | 分析 |
|------|------|
| **基本面** | 英特爾 Q1 EPS $0.29 大超預期後，AMD +13.9%、輝達 +4.3%（市值重回 5 兆美元）、台積電 ADR +5.17% 收 $402.46——這是一個板塊性的上漲，不是單一公司的利好。輝達對 OpenAI 最高 100 億美元權益投資換取未來採購 400-500 萬顆 GPU——這個數字比輝達整個季度的 GPU 交付量還大，代表輝達正在用「未來訂單」換「戰略綁定」。鴻海奪輝達 Groq 3 LPX 推論機櫃獨家供應，VR200 市占率可望提升至 55-60%——鴻海不只是一個組裝廠，正在成為輝達供應鏈的核心參與者。台積電 3 奈米及 2 奈米放量，帶動中砂、昇陽半等耗材廠擴產——耗材是景氣週期的後半段需求，當先進製程開始放量，耗材需求才會真正爆發。 |
| **技術面** | 費半 18 連漲——18 個交易日沒有實質回調，代表多頭力量極度集中。但 also 代表任何一個負面催化劑都可能引發快速止損。 |
| **國際政治** | 英特爾復甦 + AMD 大漲 = 美國半導體的集體確認，這對台灣半導體供應鏈是「客戶群擴大」的利好，而非競爭加劇的信號。 |

### 二、金融/保險

| 維度 | 分析 |
|------|------|
| **基本面** | 8 家金控股利出爐，4 檔潛在殖利率逾 4%：中信金 4.72% 最高，兆豐金、玉山金、台新新光金亦逾 4%——在中東局勢緩解後，金控股的股息吸引力再次顯現。公股銀放款中小企業金額突破 5.8 兆元（年增 2.4%），兆豐銀年增 6.45% 最突出——這代表中小企业的资本支出仍在扩张，经济的基本面没有崩。外资连 3 週回补合计买超逾 1,172 億元——这是真外资的回流，不是短线热钱。 |
| **技術面** | 高資產財管客戶數與 AUM 同創歷史新高——台灣有錢人的資產正在增加，這對財富管理業務是利好，但對社會公平是警訊。 |
| **國際政治** | 超級央行周（Fed、ECB、BOE、BOJ）——本週 Fed 利率決策將影響全球資金流向，目前市場預期全數按兵不動。 |

### 三、傳產/營建/航運

| 維度 | 分析 |
|------|------|
| **基本面** | 離岸風電 3-4 期將釋出遠超 4GW 容量供選商——這個數字較市場預期更大，代表政府在離岸風電政策上的「規模導向」而非「減量導向」。台化、台塑化正向看市況——關稅不確定因素降低、訂單動能回升，石化業加速整併有望溫和復甦。日本爆「令和時代石油危機」，建材漲價 20-30%，多地營建工程停工——這對台灣營建材料和設備出口商是意外利多（日本轉單）。 |
| **技術面** | 日本建材供應斷鏈對台灣的轉單效應可能是 Q2 的短暫利好，但 Q3 後當日本供應鏈恢復，這個效應會消失。 |
| **國際政治** | 日本「令和石油危機」的規模和 1970s 的石油危機不可同日而語——這次的供應問題是區域性的而非全球性的，但反映了中東局勢對亞洲製造業的實質衝擊。 |

### 四、網通/電信

| 維度 | 分析 |
|------|------|
| **基本面** | 波若威、華星光衝刺 CPO 光通訊商機——台積電 CPO 今年量產帶動 FAU 及 CW Laser 需求，波若威獲輝達欽點。CPO 是將光學元件（光纖、激光器、調製器）直接封裝在邏輯晶片旁邊，解決 AI 伺服器內部的高速光互聯問題，是 800G/1.6T 光收發模組之後的下一代方案。中国启动 6G 第二阶段试验、5G 用户突破 12 亿——中国的 6G 研发正在加速，但 6G 的商业化时间表仍是 2030 年前后。 |
| **技術面** | 衛星間通訊 ISL（Inter-Satellite Link）從選配升為標配——這代表低軌衛星的技術規格正在統一，對光纖雷射被動元件的需求將持續增加。 |
| **國際政治** | 中國 6G 第二階段試驗啟動——這是中美 6G 標準話語權的早期競爭，信號意義大於實質（距離商業化還有 4-5 年）。 |

### 五、生技/醫療

| 維度 | 分析 |
|------|------|
| **基本面** | 生策會會長楊泮池專訪：AI 醫療最大瓶頸在制度而非技術——這個判斷非常精準。台灣的健保給付架構沒有涵蓋 AI 輔助診斷，讓醫院使用 AI 工具沒有額外的收益誘因，技術再好也無法規模化。楊泮池提出「新藥 + AI 醫療」雙引擎戰略——這代表台灣生技業的路徑共識正在形成。 |
| **技術面** | 賴總統提四方向推動智慧醫療，健保總額首度突破 1 兆元——1 兆元的健保總額代表台灣有足夠大的醫療市場，但缺乏將科技轉化為收入的制度設計。 |
| **國際政治** | 台灣生技業的國際化（BIO Asia-Taiwan 年會）可以對冲中國生技業的價格競爭，但規模仍是劣勢。 |

### 六、綠能/電動車

| 維度 | 分析 |
|------|------|
| **基本面** | 離岸風電 3-4 期將釋出 4-6GW——这个规模是台湾之前年度规划的 2-3 倍，代表政府决心加速離岸風電的發展。但「釋出容量」不等於「併網發電」，實際建設進度取決於施工能力、供應鏈和環評。工研院指出 AI 用電劇增下，須將電力消耗轉化為智慧成長動能——這個「AI + 能源」的論述代表未來的電力需求成長將由 AI 資料中心主導，而不是傳統製造業。 |
| **技術面** | 中國河南百萬方級鹽穴儲氫示範工程投產——這是氫能基礎設施的大型個案，設備 100% 國產化代表中國在氫能技術上的自製能力正在提升。 |
| **國際政治** | 離岸風電 3-4 期的規模擴大代表台灣政府對再生能源的承諾，但執行力（施工進度、環評、海域許可）仍是主要瓶頸。 |

### 七、零售/消費

| 維度 | 分析 |
|------|------|
| **基本面** | 美國零售聯合會（NRF）發布 2026 年全球零售商 TOP50——在 2025 年宏觀環境下，「規模邏輯」遇壓，精品零售和折扣零售同時並存。「活力高雄」躋身全球隱藏瑰寶——這個國際認同對高雄的觀光品牌有長期幫助。Q1 中國消費品以舊換新銷售超 4,300 億元，服務零售額增 5.5%——中國的消費復甦比市場悲觀預期更為穩健。 |
| **技術面** | 中國服務零售額增 5.5%——這個數字顯示中國的「體驗消費」仍在成長，對台灣的觀光、餐飲業是結構性利好。 |
