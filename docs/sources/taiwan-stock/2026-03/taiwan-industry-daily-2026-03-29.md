# 台股晨報知識消化
**日期：** 2026-03-29（週末版）

## 宏觀格局判斷

1. **散戶情緒逆轉是重要的反向指標**：摩根大通數據顯示散戶 9 個月來首次淨賣出，美銀情緒指標接近「投降」階段。若過往規律有效，市場可能正在接近修正末期——但這是機率問題而非必然。

2. **川普「週末襲擊」已成為市場規律**：華爾街已識別出川普偏好週末釋出重大政策或軍事訊號，被迫在週一清晨之前調整節奏。這對主動式基金是額外的不對稱風險，對被動投資人是系統性波动來源。

3. **VIX 高檔 + 散戶淨賣出 + 內部人士逢低買入同時存在**：三者指向市場來到「多空對峙」狀態——方向不明但波動性將持續偏高，直至某個催化劑打破僵局（4/6 川普最終期限是候選人）。

## 本週關鍵事件分析

### 一、伊朗最終期限（4/6）

| 維度 | 分析 |
|------|------|
| **基本面** | 伊朗革命隊仍控制荷姆茲，每日通行船隻遠低於正常水準。貝森特聲稱「正重掌」但實際尚未恢復。若 4/6 前未達成協議，油價將面臨新一輪催化。 |
| **技術面** | 油價月線已大漲 55-57%，技術性超買明顯。但超買在黑天鵝環境中可以持續更久。 |
| **國際政治** | 波斯灣盟友私下質疑美國缺乏戰略規劃，顯示美國在中東的聯盟支撐正在松动。俄羅斯警告日本再軍事化，亞太地緣風險同步升高。 |

### 二、本週三大數據（3/31 CPI、4/3 非農）

| 維度 | 分析 |
|------|------|
| **基本面** | CPI 決定能源價格傳導是否已進入核心通膨；非農就業決定 Fed 的「雙重使命」如何平衡。若 CPI偏高 + 就業穩健，降息期望將全面消失，甚至出現定價升息。 |
| **技術面** | 市場已降低降息預期，貨幣市場估計今年升息機率達 42%。這與年初的降息狂歡形成鮮明對比。 |
| **國際政治** | 若 CPI + 非農都偏熱，Fed 可能被迫維持高利率更久，對股市和債市同時構成壓力。這是「停滯性通膨」Scenario 的關鍵變數。 |

## 跨週串聯（3/21~3/29）

此週末處於美伊戰爭的關鍵談判窗口。從 3/21 美以空襲 → 3/24 五天暫停期 → 3/26 破局 → 3/27 海峽封鎖 → 3/29 談判仍在進行，整週是一個連續升級序列。週末散戶情緒逆轉顯示市場情緒已從恐慌轉向觀望，這通常是中期底部的必要條件而非充分條件。
