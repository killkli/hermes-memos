# 台股晨報知識消化
**日期：** 2026-03-30

## 宏觀格局判斷

1. **Fed 降息預期全面消失**：鮑爾明確排除今年降息可能，市場甚至開始定價升息。這是 2024-2025 年「AI 牛市」最核心的流動性支柱被抽走。估值 compression（本益比收縮）將取代 EPS 上修成為股市的主要下行驅動。

2. **美股進入修正區間但尚非熊市**：道瓊、那指、羅素 2000 自高點回跌逾 10%，費半相對抗跌。YTD 來看，修正來得又快又急——這與 2022 年的慢性衰退模式不同，反映的是定價失誤（overreaction to rate cuts）正在被快速修正。

3. **葉門胡塞武裝正式參戰使紅海航運風險升級**：從荷姆茲單一風險擴大至「荷姆茲 + 紅海」雙重封鎖結構。全球最重要兩條能源運輸航道同時受阻，這是 1973 年石油危機以來未曾發生的結構。

## 產業深度分析

### 一、半導體/AI

| 維度 | 分析 |
|------|------|
| **基本面** | 美光自 3/18 財報後累跌 30%，輝達跌 1.4%，CoreWeave 跌 7.5%。記憶體/HBM 需求疑慮並非單一公司問題，而是整個 AI 基礎設施投資週期的階段性檢討。台積電 ADR 昨夜跌逾 1%，但費半 +6.24% 的前一交易日已建立部分緩衝。 |
| **技術面** | VIX 升破 30，代表市場恐慌情緒進入高檔。費半 +6.24% 是昨日最強指數，顯示在半導體庫存調整階段，資本支出相關個股仍是長期資金的核心倉位。 |
| **國際政治** | 西班喱關閉領空拒絕美軍使用基地，成為北約內部分歧首例。這個「盟友裂痕」信號對半導體供應鏈的象徵意義大於實質——顯示美國的聯盟體系在執行對外軍事行動時正在承受政治成本。 |

### 二、能源/傳產

| 維度 | 分析 |
|------|------|
| **基本面** | WTI 重返 $100 以上，Brent $112，月線累計大漲 55-57% 創歷史最大單月漲幅。能源股和上游原材料在這個環境下直接受益，但這是「成本推動」而非「需求拉動」，對中下游製造業是淨利空。 |
| **技術面** | 油價技術性超買，但黑天鵝期間超買可以持續。月線大漲背後是供給中斷的實質威脅，而非投機情緒。 |
| **國際政治** | 貝森特聲稱「正重掌荷姆茲」，已有更多船隻開始通行——這是美國試圖管理市場預期的說法，每日 1,000-1,200 萬桶缺口尚未完全解決。 |

### 三、金融/保險

| 維度 | 分析 |
|------|------|
| **基本面** | 鮑爾談話試圖安撫市場但已排除降息，10 年期美債殖利率下滑 10bp。殖利率下滑代表市場認為 Fed 的緊縮效果將開始顯現——但這也可能預示經濟放緩。 |
| **技術面** | 美股進入修正區間的定義（自高點回跌逾 10%）已觸發部分量化基金的風險減持信號，可能加速拋售。 |
| **國際政治** | Fed 的「通膨預期仍穩定」說法試圖將戰爭定義為「供給面衝擊」而非結構性通膨。若這個判斷被事實推翻（3/31 CPI 顯示核心通膨上升），Fed 將面臨更大的政治壓力。 |
