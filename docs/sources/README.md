# Raw Sources

Immutable source documents — articles, papers, data files, transcripts.

**Rule:** LLM reads but **never modifies** these files. All synthesis and analysis happens in wiki pages.

## Subdirectories

- `articles/` — Web articles, blog posts, news
- `papers/` — Academic papers, arxiv, technical reports
- `transcripts/` — Meeting notes, interviews, lecture notes
- `data/` — Data files, datasets, statistics
- `assets/` — Images, diagrams referenced by sources
