# Synthesis Pages

Cross-topic synthesis and overview pages that combine knowledge from multiple entities and concepts.

## Types of Synthesis

- **Comparisons** — Side-by-side analyses of related topics
- **Overview Pages** — Broad summaries of domains or themes
- **Timeline Pages** — Chronological development of ideas
- **Relationship Maps** — How entities and concepts connect
- **Filed Query Results** — Substantial answers worth preserving

## Page Format

Every synthesis page should include:
- What is being synthesized and why
- Key insights from the synthesis
- Connections to source entities and concepts
- Source references
